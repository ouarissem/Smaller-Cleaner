# Windows Cleaner by Waresem

A simple and safe Windows cleaning tool that removes temporary files, logs, cache, and other junk to free up space and improve system performance.  
The tool automatically runs as Administrator and performs a fast cleanup without harming any system files.

---

## ⭐ Features
- Auto-run as Administrator  
- Clean User Temp  
- Clean System Temp  
- Clean Prefetch  
- Clear Windows Update Cache  
- Remove Logs & Crash Dumps  
- Clean Delivery Optimization Cache  
- Empty Recycle Bin  
- Safe for all Windows versions

---

## 📦 How to Use
1. Download the `.bat` file  
2. Double-click to run  
3. The cleaner will ask for admin permission automatically  
4. Wait for the process to finish

---

## 📁 File Included
- `windows-cleaner.bat`

- PowerShell (built-in)
