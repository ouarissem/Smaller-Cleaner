## 📥 Installation & Usage

### Quick Start
1. Download `Cleaner_PRO_2026.bat`
2. Right-click → Run as Administrator
3. Select your preferred cleaning mode
4. Wait for completion
5. System restart (optional but recommended)

### Interactive Menu
```
Main Menu Options:
[1] Quick Clean     - Fast and safe cleanup (2-3 minutes)
[2] Advanced Clean  - Deep system optimization (5-10 minutes)
[3] Selective Clean - Choose specific tasks
[4] Browser Cleanup - Clear browser caches
[5] Registry Optimization - System optimization
[6] Schedule Auto Cleanup - Daily automatic cleanup
[7] View Cleanup Log - See cleanup history
[8] Exit - Close the program
```

### Detailed Cleaning Operations

#### 1. Quick Clean
Cleans essential temporary files:
- User Temp folder
- System Temp folder
- Prefetch cache
- Windows Update cache
- Logs and Crash Dumps
- Recycle Bin

#### 2. Advanced Clean
Includes all Quick Clean tasks PLUS:
- Delivery Optimization cache
- Windows component store
- DNS cache
- Error reporting files
- Additional temporary caches

#### 3. Selective Clean
Choose exactly what to clean:
- Temp files only
- Cache files only
- Logs only
- Recycle Bin only
- All of the above

#### 4. Browser Cleanup
Removes cache from:
- Google Chrome
- Mozilla Firefox
- Microsoft Edge

#### 5. System Optimization
- Registry cleanup
- Disk defragmentation
- Storage optimization

#### 6. Auto-Scheduler
Creates a Windows scheduled task for:
- Daily automatic cleanup
- Runs at 2:00 AM
- Requires Administrator privileges
- No user interaction needed
